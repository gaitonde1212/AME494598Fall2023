Temperature/Humidity

Tutorial

  • Step 1: Login Your AWS Account,Open Sandbox and 'start lab'

  • Step 2: Select EC2

  • Step 3: Launch an Instance

  • Step 4: We will be using Ubuntu Version 20.04

  • Step 5: Select the Key and the Storage

  • Step 6: Launch Instance

  • Step 7: Click on your instance ID

  • Step 8: Go to Security

  • Step 9: Edit the inbound Rules

  • Step 10: Connect to your instance

  • Step 11: Install Node/Mongo on EC2.

    use command: curl -s https://gist.githubusercontent.com/tejaswigowda/f289e9bff13d152876e8d4b3281142f8/raw/df37df2e16a3166e686357a045751a7c18bbeebe/installNodeMongoUbuntu16.04.sh | bash

  • Step 12: git clone 'the repository'

    Go to the following folder >watchSensors/captureDataServer-TempHum

  • Step 13: node server.js

  • Step 14: Now on Arduino IDE, Program the Watch connected to Temperature and Humidity Sensor

  • Step 15: You will see the data on your HTML link

  • Step 16: Now for For Mongo we will install some libraries

    npm install mongoskin@2

    npm install mongodb@2.2

  • Step 17: Make sure to configure your mongod

    sudo nano /etc/mongod.conf

    I have set my bindIP as 0.0.0.0

  • Step 18: Start the service for mongodb

    sudo service mongod start

  • Step 19: Connect to EC2 IP

  • Step 20: We can see a database created as 'sensorData'

  • Step 21: We can see the data on Mongodb
